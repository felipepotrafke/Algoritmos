# Exercícios de Lógica de Programação – Visualg

Este repositório reúne exercícios de **lógica de programação** desenvolvidos em **Visualg**, como parte do meu processo de aprendizado em programação.

Os exercícios foram feitos com base no conteúdo do curso de **Algoritmos** do **Curso em Vídeo**, ministrado pelo professor Gustavo Guanabara, servindo como prática e aprofundamento dos conceitos apresentados nas aulas.

O objetivo deste repositório é demonstrar:
- raciocínio lógico
- uso correto de estruturas de decisão e repetição
- controle de variáveis, contadores e acumuladores
- identificação e correção de erros lógicos
- evolução gradual na forma de pensar algoritmos

---

## Conteúdos abordados

Os exercícios estão organizados por **conceito**, e não apenas por numeração:

- **Estruturas condicionais**
  - `se`, `senao`, decisões encadeadas
- **Laços de repetição**
  - `enquanto`, `repita...ate`, `para`
- **Contadores e acumuladores**
  - soma, média, maior e menor valor

---

## Evolução do aprendizado

Os exercícios foram desenvolvidos de forma **progressiva**, acompanhando o avanço do conteúdo estudado.

---

## Tecnologias e conceitos utilizados

- Visualg
- Pseudocódigo estruturado
- Lógica de programação (base conceitual para qualquer linguagem)

---

## Objetivo

Este repositório funciona como:
- registro do meu aprendizado em lógica de programação
- base sólida para evolução em linguagens como C, Python ou Java
- evidência de disciplina, consistência e desenvolvimento do raciocínio lógico

---

## Observação

Os algoritmos aqui não têm foco em uso em produção.

O foco é **aprender a pensar corretamente antes de codar**, priorizando:
- clareza
- lógica
- evolução contínua
